The Big Idea: A Merchant Onboarding Intelligence Agent
Instead of a consumer underwriting pipeline, flip the perspective to the B2B layer — which is actually Fragile's core business.

Here's what it does:
A merchant (say, a consumer electronics brand) submits a request to offer device financing. The agent autonomously:

Ingests the merchant's device catalog — scrapes or accepts a CSV of devices, MSRPs, and expected depreciation curves
Runs a dbt pipeline that models risk parameters per device category — expected residual value, historical default rates by device type, lease term recommendations
Calls an LLM via API to generate a merchant-specific financing proposal — recommended lease terms, pricing bands, risk tier thresholds
Emails a structured onboarding report back to the merchant with suggested product configurations
